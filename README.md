# Backend Development Roadmap Website

A simple multi-page learning website that provides a **structured roadmap for learning Backend Development using Python and Databases**.
The site is built using **Python (Flask), HTML, CSS, and JavaScript** and includes multiple pages with curated learning resources.

---

## 📌 Features

* Multi-page website
* Backend built with **Flask**
* Structured learning roadmap
* Links to tutorials and documentation
* Reusable page templates
* Basic frontend styling and JavaScript interactivity

---

## 🛠️ Technologies Used

* **Python**
* **Flask**
* **HTML5**

## ⚙️ Installation

1. Clone the repository

```
git clone https://github.com/yourusername/backend-roadmap-site.git
```

2. Navigate to the project folder

```
cd backend-roadmap-site
```

3. Create a virtual environment (optional but recommended)

```
python -m venv venv
```

Activate it:

**Windows**

```
venv\Scripts\activate
```

**Mac/Linux**

```
source venv/bin/activate
```

4. Install dependencies

```
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Flask development server:

```
python app.py
```

Open your browser and go to:

```
http://127.0.0.1:5000
```
